## ğŸ“Œ Project Overview
**Perfume Finder** is a sophisticated web application designed to navigate the complex world of fragrances. Moving away from standard lists, this tool offers a **"Luxury Tech" interface** that allows users to filter thousands of perfumes based on olfactory notes, gender categories, and community ratings.

The project demonstrates the integration of **Data Analysis** (Python/Pandas) with **Web Development** (Streamlit) and **Digital Analytics** (GA4).
